# 🌙 Dream Journal Analyzer

An AI-powered web application that analyzes dream descriptions and provides meaningful interpretations using **Google Gemini AI** and **LangGraph**. The application features a modern web interface with a Flask backend for seamless dream analysis.

---

## 📖 Overview

Dream Journal Analyzer helps users understand the possible meanings behind their dreams. Users simply enter a dream description, and the AI generates an insightful interpretation using Google's Gemini model.

---

## ✨ Features

* 🌙 AI-powered dream analysis
* 🤖 Google Gemini AI integration
* 🧠 LangGraph multi-agent workflow
* ⚡ Flask REST API backend
* 🎨 Modern and responsive user interface
* 🔒 Secure API key management using `.env`
* 💬 Instant dream interpretation

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/kn675818-stack/dream-journal-analyzer.git
cd dream-journal-analyzer
```

### Create a virtual environment

```bash
python -m venv .venv
```

### Activate the virtual environment

**Windows**

```bash
.venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file and add your Gemini API key:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
