# 🌙 Dream Journal Analyzer

An AI-powered web application that analyzes dream descriptions and generates meaningful interpretations using **Google Gemini AI** and **LangGraph**. Built with a Flask backend and a responsive web interface, the application demonstrates the integration of Large Language Models (LLMs) into a real-world AI application.

---

## 📖 Overview

Dream Journal Analyzer enables users to explore the possible meanings of their dreams through AI-powered analysis. Users simply enter a dream description, and the system processes it using **Google Gemini AI** to generate thoughtful and insightful interpretations.

---

## ✨ Features

* 🌙 AI-powered dream interpretation
* 🤖 Google Gemini AI integration
* 🧠 LangGraph multi-agent workflow
* ⚡ Flask REST API backend

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/kn675818-stack/dream-journal-analyzer.git
cd dream-journal-analyzer
```

### Create a Virtual Environment

```bash
python -m venv .venv
```

### Activate the Virtual Environment

**Windows**

```bash
.venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file in the project root and add your Gemini API key:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run the Application

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```
